美国旅行指南 —— 涵盖5大区域、12个目的地城市、3条自驾路线、季节性旅行建议、文化亮点，以及实用的行程规划要点。

**在线体验：** https://tgthms.github.io/usa-travel-guide/

## 功能特色
- 🌎 **全美旅行覆盖** 涵盖美国全部 50 个州，并提供分类整理的区域信息。
- 🧭 **区域化架构** 内容按主要地理区域（如东北部、南部、中西部、西部和西南部）进行组织。
- 📸 **精选目的地内容** 收录全美各地的著名地标、城市及风景名胜。
- 🌦️ **季节性旅行建议** 推荐游览不同区域及目的地的最佳时机。
- 🧰 **旅行实用工具** 快速处理常见旅行需求：实时汇率换算（使用Frankfurter API）、时区查询及餐厅账单计算。
- 🌐 **多语言支持** 支持英语、西班牙语、中文和日语，满足更广泛用户的需求。
- ⚙️ **个性化设置** 允许用户自定义语言和度量单位等偏好设置。
- ♿️ **辅助功能** 可减弱动态效果或关闭光标特效——适合对动态内容敏感的用户、老旧设备，或追求极简视觉体验的用户。
- 🗂️ **结构化内容布局** 信息组织井井有条，便于浏览与探索。

> *更多功能即将推出。*
