# 美国旅行指南

一个单页、动画丰富的美国旅行指南 —— 涵盖5大区域、12个目的地城市、3条自驾路线、季节性旅行建议、文化亮点，以及实用的行程规划要点。

**在线体验：** https://tgthms.github.io/usa-travel-guide/

## 功能特色
- 6种视觉主题
- 支持3种语言（英语、中文、日语）
- 可切换单位（°F/°C、英里/公里）—— 设置会自动保存
- 采用单一自包含 HTML 文件构建，除 Google Fonts 外无其他外部依赖

*未来将添加更多目的地、国家及功能。*
