## ✨ 功能特色

| | |
|---|---|
| 🌍<br>**全美<br>覆盖** | 全部 50 州，按东北、南部、中西部、西部、西南等区域排序。 |
| 🏙️<br>**推荐<br>城市** | 十二座城市卡片、评分与详情。 |
| 🔗<br>**实用<br>外链** | 官方旅游、公共交通、公园与博物馆链接。 |
| 🖼️<br>**相册** | 可筛选的旅行相册：搜索、排序、画质（缩略图 / 中等 / 原图）*支持 HDR* |
| 🧰<br>**旅行<br>工具** | 汇率、世界时钟、小费与销售税（50 州）、油车/电车费用、紧急电话。 |
| 🌦️<br>**季节<br>文化<br>与<br>路线** | 何时出行、文化亮点与经典公路。 |
| 🎲<br>**美国<br>趣味<br>冷知识** | 每次随机一条，多语言支持 |
| 🌐<br>**多语言** | 英 / 西 / 中 / 日（含法律页）。 |
| ⚙️<br>**个性化** | 主题、语言、华氏/摄氏与英里/公里、相册画质——保存在本机。 |
| ♿️<br>**无障碍** | 三级动效：完整效果、减弱动态效果，或关闭动态效果。对于桌面端，光标拖尾可单独控制。 |
| 📱<br>**移动端<br>友好** | 轻量首屏、触控即时滚动与性能优化。 |
| 🔒<br>**隐私<br>与<br>条款** | 站内[隐私政策](../../privacy.html)与[使用条款](../../terms.html)，含外链说明。（法律文件的翻译仅供参考，以英文版为准） |
